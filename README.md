# 🍫 Chocolate Sales Data Warehouse

## 📌 Project Overview

This project demonstrates the design and implementation of a Data Warehouse for Chocolate Sales data using an ETL pipeline built with Pentaho Data Integration (PDI) and PostgreSQL. The project transforms raw transactional data into a structured data warehouse to support business analysis and decision-making.

---

## 🎯 Objectives

- Build an end-to-end ETL pipeline.
- Design a Star Schema Data Warehouse.
- Clean and transform raw sales data.
- Create interactive dashboards for business insights.
- Support data-driven decision making.

---

## 🔄 ETL Workflow

1. Extract raw Chocolate Sales data.
2. Transform data by cleaning and validating records.
3. Load data into a PostgreSQL Data Warehouse.
4. Build Star Schema.
5. Visualize business insights using Tableau.
